## 제약사항

- js의 html코드는 아래로 고정 (html+css로만 구현하는거 방지, 우리는 CSR을 연습하려고 함)
- px쓰지말것

```
<!doctype html>
<html lang="ko">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>넷플릭스 클론</title>
  </head>
  <body>
    <noscript>You need to enable JavaScript to run this app.</noscript>
    <div id="root"></div>
  </body>
</html>
```
